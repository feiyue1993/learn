# 快捷键

- command+shift+p 快速调出命名面板
- command+s       保存当前文件
- command+shift+s 另存为
- command+alt+s   全部保存
- command+\       切换目录树状态
- control+0       聚焦目录树
- command+p       在当前文件搜索
