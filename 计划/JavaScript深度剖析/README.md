#   ECMAScript新特性
### ECMAScript新特性
-   JavaScript vs ECMAScript
```
   JavaScript: 是一种脚本语言
   ECMAScript: 是一种语言规范
``` 


-   块作用域和模版字符串
```
    块作用域:   
        什么是作用域
        答:规定了如何查找变量,确定当前代码对变量的访问权限。
        
        拓展
        静态作用域(即词法作用域,js使用静态作用域):静态作用域中的函数遇到既不是形参，也不是函数内部定义的局部变量时，会去函数定义时的环境中查找.
        动态作用域:动态作用域中的函数遇到既不是形参，也不是函数内部定义的局部变量时，会去函数调用的环境中查找.
        
        什么是块级作用域
        答：在任何一对花括号中的语句集都属于一个块，在这里定义的所有变量在代码外面是不可见的.
        
    模版字符串(反钩号):
        -   表达式
        -   多行字符串
        -   标签模版字符串
        -   原始字符串
        
        使用
            -   `内容`
            -   ${}识别储存的内容片段
            -   tag `内容` 触发高级函数的定义
            -   String.raw `` 取得字符串模版的原始数据
            
```
